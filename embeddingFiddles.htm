Встраивание проектов

Встраивание предназначено для показа проекта с работающим результатом на любой другой странице.

Встраивание доступно из действующего меню, пункт ‹› Embed.

Вставка имеет несколько вариантов отображения:

Tabs – выбор вкладок, которые будут показаны во вставке. Порядок расположения вкладок (js,css,html,results) можно изменить в строке URL-адреса - смотрите ниже.
Если отключить все вкладки, то будут показаны все.
Если в проекте были подключены внешние ресурсы, они также будут показаны: вкладка Resources. При отключении любой из вкладок (JavaScript, HTML, CSS, Result), вкладка ресурсов также отключается.
Вкладка ресурсов выглядит примерно так:

External resources loaded into this fiddle:
three.min.js

то есть, в ней сообщается, что в данном проекте загружен внешний ресурс и дается название этого ресурса, в нашем случае это библиотека three.min.js. При этом название ресурса будет ссылкой на загруженный файл.

Visual – светлая (Light) или темная (Dark) тема оформления.
Код вставки фрейма со светлой темой (см. примечание переводчика ниже):
<iframe src="https://jsfiddle.net/zalun/NmudS/embedded/result,js,html,css/"></iframe>
Код того же фрейма, но с темной темой будет таким: 
<iframe src="https://jsfiddle.net/zalun/NmudS/embedded/result,js,html,css/dark/"></iframe>
Также можно установить конкретные варианты цвета (шестнадцатиричный код цвета):
Font color - цвет шрифта,
Accent color - цвет выделения,
Code background - фон кода,
Menu background - фон меню.
Примечание переводчика: Все эти дополнительные настройки цвета (Font color, Accent color, Code background, Menu background) не работают. Код для вставки, как скрипта, так и фрейма изменяется, в нем появляются соответствующие параметры, но изменений не происходит.
Например, код скрипта:
<script async src="//jsfiddle.net/zalun/NmudS/embed/?fontColor=ff0000&accentColor=ff0000&bodyColor=0000ff&menuColor=0000ff"></script>
код фрейма
<iframe width="100%" height="300" src="//jsfiddle.net/zalun/NmudS/embedded/?fontColor=ff0000&accentColor=ff0000&bodyColor=0000ff&menuColor=0000ff" allowfullscreen="allowfullscreen" allowpaymentrequest frameborder="0"></iframe>
Примечание переводчика: Обратите внимание на сам код, что в скрипте, что во фрейме, в значении аргумента src пропущено начало url-адреса, а именно: https:. Без этого участка кода вставка не будет работать и его нужно добавить вручную.

Embed snippet – для вставки можно использовать либо iframe, либо script.
Для перехода к iframe, правее названия поля Embed snippet есть ссылка-вопрос Prefer iframe?: - предпочитаете фрейм? При клике на ней произойдет смена кода скрипта на код фрейма, а ссылка-вопрос изменится на Use script - использовать скрипт, клик по ней вернет код скрипта обратно.
Преимущество использования скрипта в том, что при этом не блокируется отображение и высота может задаваться автоматически, так что он соответствует коду в каждой вкладке.
При переходе к iframe ниже кода фрейма дается предупреждение об этом:
No autoresizing to fit the code
Render blocking of the parent page
что означает:
Нет автоматического изменения размера окна в соответствии с размером данного кода
Блокировка визуализации родительской страницы

Изменение порядка расположения вкладок

Например, чтобы показать вкладку результата первой добавьте в строку URL-адреса result,js,html,css - вот так: 
<iframe src="https://jsfiddle.net/zalun/NmudS/embedded/result,js,html,css/"></iframe>
