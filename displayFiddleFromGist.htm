Отображение проекта JSFiddle из Gist

Демонстрация считывается из Github Gist и представляется как проект.

Структура файлов Gist


fiddle.js
fiddle.html
fiddle.css
fiddle.manifest

Название поля
File name		Описание
Description
fiddle.[js/html/css]содержит код проекта указанной панели.
fiddle.manifestYAML description of the Gist for JSFiddle to parse


Пример файла manifest ()


name: The Name of the Fiddle (название проекта JSFiddle)
description: Some description, please keep it in one line (некое описание, следует уложиться в одну строку)
authors:
  - John Doe
  - Jan Wisniewski
resources:
  - http://some.url.com/some/file.js
  - http://other.url.com/other_filename.css
normalize_css: no
wrap: b
panel_js: 1
panel_css: 1

 ⓘ   Все поля необязательны.

Поля файла manifest

panel_html – язык HTML панели. Допускаемые возможные значения:
      0 – HTML

panel_css – язык CSS панели. Допускаемые возможные значения:
      0 – CSS
      1 – SCSS

panel_js – язык JS панели. Допускаемые возможные значения:
      0 – JavaScript
      1 – CoffeeScript
      2 – JavaScript 1.7

resources – список внешних ресурсов.

name – название проекта.

description – описание проекта.

normalize_css – Normalize CSS by loading normalize.css before any CSS declarations.
      yes – пормализовать
      no – не нормализовывать

wrap – Set the JS code wrap. Options:
      l – On load
      d – On DOM ready
