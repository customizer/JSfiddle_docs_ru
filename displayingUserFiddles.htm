Отображение проектов пользователя

Имеется возможность создания плагинов (модулей) отображения проектов, созданных конкретным пользователем.
метод GET 
https://jsfiddle.net/api/user/:username/demo/list.json
 ⓘ   Пример:
http://jsfiddle.net/api/user/zalun/demo/list.json?callback=Api&sort=framework&start=5&limit=5

      Запрос (Request)     Ответ (Response)
Параметры запроса (Query Parameters)
callback
(дополнительный,
необязательный
параметр)		stringИмя функции для перекрестного домена. Если функция обратного вызова не предоставлена, возвратится стандартный JSON.

Значение по умолчанию: нет.
start
(дополнительный,
необязательный
параметр)		integerСмещение элемента.

Значение по умолчанию: 0.
limit
(дополнительный,
необязательный
параметр)		integerКоличество возвращаемых элементов.

Значение по умолчанию: 10
sort
(дополнительный,
необязательный
параметр)		stringТип сортировки.
По дате (date), алфавиту (alphabetical) или фреймворку (framwork)
Значение по умолчанию: date.
order
(дополнительный,
необязательный
параметр)		stringdesc или asc

Значение по умолчанию: asc.
framework
(дополнительный,
необязательный
параметр)		stringФильтр фреймворка.

Значение по умолчанию: нет.
