Передача ответа непосредственно из архива (репозитория) Github

В то время как демонстрация может использовать асинхронные запросы, JSFiddle будет считывать их и отправлять с помощью правильного MIME-типа.

Структура каталога демонстрации


Demo Directory/
  demo.response.html
  demo.response.json
  demo.response.xml

Запуск демонстрации в JSFiddle


https://jsfiddle.net/gh/get/response.{extension}/{github_tree}/

Части URL-адреса в примерах:
extension – расширение файла demo.response.{extension} в репозитории демонстрации на github'е.
github_tree – путь к каталогу, который используется для его отображения на странице github'а (без http://github.com/).
Работоспособность JSONP не предоставляется. Её можно получить при помощи необработанного URL-адреса репозитория GitHub'а.
