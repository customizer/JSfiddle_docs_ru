Отображение проекта JSFiddle из архива (репозитория) Github

Это позволяет опустить часть размещаемого кода на JSFiddle и загрузить его из архива (репозитория) Гитхаб'а непосредственно в редактор.

Демонстрация структуры каталога


Demo Directory/
  demo.js
  demo.html
  demo.css
  demo.details

demo.[js|html|css] - содержат код проекта для указанных панелей.
demo.details - описание демонстрации на языке YAML.

---
 name: Name of the Demo (название демонстрации)
 description: Some description, please keep it in one line (некое описание, следует уложиться в одну строку)
 authors:
   - John Doe
   - Jan Wisniewski
 resources:
   - http://some.url.com/some/file.js
   - http://other.url.com/other_filename.css
 normalize_css: no
 load_type: d
 ...

 ⓘ   Все поля необязательны.

Запуск демонстрации в JSFiddle

Части URL-адреса в примерах:
framework – требуемый фреймворк для загрузки вместе с кодом из архива
version - часть строки версии фреймворка - будет использоваться последний проход поиска. Если будет задано 1.3, JSFiddle будет использовать последний результат поиска, т.е. будет отдано предпочтение версии 1.3.2 относительно 1.3.1 и 1.3 версий.
dependency_list – список зависимостей в виде частей строки, разделенных запятой. Здесь будет отмечена любая зависимость, содержащую подстроку. It would mark any dependency containing the substring
github_tree – путь к каталогу, используемому для его отображения на странице github (без http://github.com/)

Демонстрация + отсутствие JS фреймворка

При отсутствии фреймворка и версии используется library/pure: 

https://jsfiddle.net/gh/get/library/pure/{github_tree}/

 ⓘ   Пример: https://jsfiddle.net/gh/get/library/pure/{github_tree}/

Демонстрация + JS фреймворк с версией

Демонстрация с предустановленным фреймворком (например, jQuery 3.0): 

https://jsfiddle.net/gh/get/{framework}/{version}/{github_tree}/

 ⓘ   Пример: https://jsfiddle.net/gh/get/jquery/3.0/{github_tree}/
Данная демонстрация получена из следующего архива (репозитория):
https://github.com/zalun/jsFiddleGithubDemo/tree/master/Demo/

Демонстрация + JS фреймворк с версией + зависимости

Демонстрация с фреймворком и внешними ресурсами/зависимостями: Demo with a framework and external resources/dependencies 

https://jsfiddle.net/gh/get/{framework}/{version}/dependencies/{dependency_list}/{github_tree}/
