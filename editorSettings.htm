Настройки редактора

Кнопка вызова настроек Settings находится справа на панели действий.

Верхнее поле в окне настроек отвечает за настройку макета рабочей области JSFiddle - Editor layout (Макет редактора). Значение по умолчанию - Classic, классический макет, представляющий собой сетку 2×2. Подробнее смотрите ниже.

На втором поле - общие настройки редактора - General. Здесь можно выбрать:
тему внешнего вида редактора (Dark theme - темная тема (значение по умолчанию), если этот пункт не включить будет светлая тема - Light),
указать или не указывать номера строк кода (Line numbers) - по умолчанию включено, т.е. номера строк будут показаны,
переносить или нет длинные строки (Wrap lines) - по умолчанию длинные строки переносятся,
Indent with tabs - отступы с табуляцией. По умолчанию отключено.
Code hinting (autocomplete) (beta) - подсказки кода и автозавершение (как пробная версия). По умолчанию отключено.

Также здесь есть три параметра по оформлению кода:
Indent size - размер отступа слева, есть три значения:
2 spaces - т.е. отступ в 2 символа (значение по умолчанию),
3 spaces - 3 символа и
4 spaces - 4 символа.
Key map - карта команд с клавиатуры
Default - т.е. значение по умолчанию,
Sublime Text - в стиле текстового редактора Sublime Text,
EMACS - в стиле текстового редактора EMACS.
Font size (размер шрифта), можно выбрать одно из следующих значений:
Default - т.е. значение по умолчанию,
Big - крупный шрифт,
Bigger - большущий шрифт (крупнее чем Big) и
Jabba - самый крупный шрифт.

Третье поле - Behavior (поведение редактора). Здесь имеются следующие настройки:
Auto-run code - автоматический запуск кода на выполнение. По умолчанию отключен.
Only auto-run code that validates - автоматический запуск на выполнение только проверенного кода. По умолчанию отключен.
Auto-save code (bumps the version) - автоматическое сохранение кода (увеличивает номер версии). По умолчанию отключено.
Auto-close HTML tags - автоматическое закрытие HTML-тегов. По умолчанию включено.
Auto-close brackets - автоматическое закрытие скобок. По умолчанию включено.
Clear console on run - очистка консоли при запуске кода. По умолчанию отключено.
Live code validation - проверка кода во время его написания. По умолчанию включено.
Highlight matching tags - выделение парных HTML-тегов (подчеркиванием). По умолчанию отключено.


Последнее, самое нижнее поле отвечает за панель шаблонов - Boilerplates. В ней всего один пункт - Show boilerplate bar less often (Показывать панель шаблонов пореже).

Макеты и темы

JSFiddle позволяет переключать раскладку редактора с классической сетки 2×2 на то, что соответствует вашим потребностям.
Размеры каждой ячейки сетки можно изменять по ширине и высоте. Кроме того, при изменении размера панели результата (Result) показывается её ширина в пикселах (удобно для работы с медиа запросами).
