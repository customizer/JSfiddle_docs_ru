Отображение проекта по методу POST

Имеется возможность показать проект, который не сохранен в JSFiddle, при помощи метода POST из стороннего источника.

Структура URL-адреса

framework – название нужного фреймворка. Каковой фреймворк должен быть загружен вместе с проектом.
version – часть строки версии фреймворка - будет использоваться последний проход поиска. Если будет задано 1.3, JSFiddle будет использовать последний результат поиска, т.е. будет отдано предпочтение версии 1.3.2 относительно 1.3.1 и 1.3 версий.
dependency_list – части строк, составляющих список зависимостей, разделенные запятой.

Фреймворк

Загружаемый фреймворк (или его отсутствие) должен быть указан в строке URL-адреса, например: 
jsfiddle.net/api/post/{framework}/{version}
Если фреймворк не используется, следует указать library/pure, например: 
jsfiddle.net/api/post/library/pure/

Зависимости

Существует возможность добавления зависимостей в виде списка зависимостей, разделенных запятой: 
jsfiddle.net/api/post/{framework}/{version}/dependencies/{dependency_list}/

Переменные POST

Все поля дополнительны, необязательны. Если никаких данных не предоставлено, будут использоваться настройки нового проекта.
метод POST ​ 
https://jsfiddle.net/api/post/:framework/:version/dependencies/:dependency_list/

wrap
(дополнительный,
необязательный
параметр)		stringУстанавливает упаковку JS-кода: Возможные значения:
- l код упаковывается так, что он будет запущен по событию окна onload, т.е. когда загрузилась вся страница, включая стили, картинки и другие ресурсы.
- d код упаковывается так, что он будет запущен по событию окна onDomReady, т.е. когда браузер полностью загрузил HTML, было построено DOM-дерево, но внешние ресурсы, такие как картинки <img> и стили, могут быть ещё не загружены.
- h код JavaScript не упаковывается и размещается внутри и внизу тега <head>, т.е. пока не будет выполнен скрипт, содержимое страницы не будет показано.
- b код JavaScript не упаковывается и размещается внутри и внизу тега <body>, т.е. вначале показывается страница, а затем выполняется скрипт.
